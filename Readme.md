# Клиент
Данные клиент предназначен для работы в совокупности с [сервером](https://github.com/NoTimeToPeace/QtTcpServer).

## Функционал:
- Подключение к серверу;
- Уведомление о подключении к серверу;
- Отправка сообщений на сервер.
